Notation

A short form of writing a⋅a⋅a is a3 . The letter a is called the base while the number 3 is the power, index or exponent


don't worry you'll get to know all this in few seconds

Laws of Indices

Note: we're using dot notion as multiplication so don't be confused i.e ⋅ = *

  • Law of Multiplication

    a2 = a⋅a
    Examples

    2⋅ 2 ⋅ 2 = 23 = 2 x 2 x 2 = 8

    Solve
    23

    Solve
    2 x 2 x 2

    In General

    am x an = am+n
    Examples

    23 ⋅ 25 = 23+5 = 256

    Solve :

    23 x 25

    32 x 37

    Multiplication of different bases

    42 x 23
    solve simultaneously thus,
    42 = 16
    then 23 = 8
    16 ⋅ 8 = 128

  • Law of Division

    am / an = am-n
    a4 / a2 = a4-2
    Examples

    124 / 42

    ... Firstly divide the base
    ... Secondly subtract the powers from each other.

    i.e (12/4)4 - 2
    32

    Solve
    23 / 1

    Solve
    242 / 24

  • Law of Reciprocal

    a-1 = 1/a

    Example

    4-2 will be solved as this

    1/(4)2 = 1/16

  • Zero exponent

    Note : if any figure is with the exponent zero then the value is said to be one

    Example

    n0 = 1

Let's Recap

ax x ay = a8.......Law of Multiplication

an / am = a3-2.......Law of Division

a-m = m/a.......Reciprocal Law

a0 = 1 .......Zero Exponent

Solved questions

1. Simplify this (27)2/3

2. Solve this(1/8)x = 8

3. 10-x = 0.000001

4. Solve 3(3)x = 27

5. 52x - 26(5x) + 25 = 0

Solutions

1. (27)2/3
(33)2/3.... this is the breaking point
(33)2/3....Now we're left with exponential 2
i.e 32 = 9

2. (1/8)x = 8
we use the law of reciprocal here..
thus(8-x) = 8
we reduce both side by 8
8-x = 81
hence... -x = 1 therefore x = -1

3. 10-x = 0.000001
10-x = 10-6
we also reduce... both side by 10
10-x = 10-6 -x = -6
hence... x = 6

4. 3(3)x = 27
3x + 1 = 33
we also reduce... by 3 3x + 1 = 33
x + 1 = 3
therefore... x = 3 - 1
x = 2

5. 52x - 26(5x) + 25 = 0
Here is a quadratic form... let 5x = p
p2 - 26p + 25 = 0
Let's factorise...
p2 - 25p - p + 25 = 0
(p2 - 25p) - (-p + 25) = 0
p(p - 25)(p - 25) = 0
(p - 25)(p - 1) = 0
hence... p = 25 or p = 1
thus, p = 5x, let p = 25
25 = 5x, 52 = 5x
x = 5 ,when p = 25.

let p = 1 , where p = 5x
Now the Zero exponent i.e X0 = 1 , 50 = 5x
reduce both side by 5... x = 0,
hence... when p = 1, x = 0
